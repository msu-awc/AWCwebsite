Software Resources

Software and Sites We Love

Frequently Used by CS Majors
Bitvise - SSH software (Windows Only)
Git - version control for all your projects
GitHub or GitLab - web-based version-control and collaboration platforms for software developers
StackOverflow - online community for developers to learn and share​ ​their programming ​knowledge
Visual Studio Code - code editor with lots of extensions and plugins 
Notepad++ - powerful text editor (Windows Only) 
Sublime Text - a sophisticated text editor
IntelliJ IDEA or Eclipse or NetBeans - IDEs for developing software
Unity - game development software
Ubuntu - open source software operating system

Helpful for Students
Chrome or Firefox - popular Internet Browsers
            Please avoid Internet Explorer or Microsoft Edge
LibreOffice - an open-source Microsoft Office alternative
Overleaf - online LaTeX editor 
SumataraPDF - view multiple types of documents

Imaging, Film, and Audio
Krita - open source painting program
ShareX - Screen capture/record
GIMP - image editor
DaVinci Resolve - video editor
VLC - multimedia player
OBS - video recording/live streaming software

Communication
Discord - text, voice, and video chatting
            Email us to join the MSU Computer Science Server, just for students!
Zoom - video/web conferencing often used by professors

Computer and File Management
7-Zip - An archive manager
Ninite - Install and Update All Your Programs at Once
Patch My PC - software to keep your programs up to date
CCleaner -  system optimization, privacy and cleaning tool
