Hour of Code

Showing everyone can code

PreviousNext

Hour of Code is a global movement designed to demystify the art of computer science, showing anyone can learn the basics. Twice a year, we host an Hour of Code to students K-12 in the area to learn how to code in an exciting manner.

Keep a lookout for volunteer signups in your email! For more information, contact our officers or Sharlyn at sharlyn.izurieta@montana.edu.
